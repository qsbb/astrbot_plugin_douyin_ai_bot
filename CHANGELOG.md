# 更新日志

## v1.0.0 (2026-07-10)

### 初始版本
- 抖音评论自动回复，LLM 生成回复内容
- 好感度系统（6 个等级：主人/好友/熟人/粉丝/陌生人/厌恶）
- 每日心情系统 + 节日彩蛋
- 语义记忆（对话存储 + 关键词检索注入）
- 黑名单管理
- 主动刷视频（定时调度 + LLM 评分 + 点赞/评论/关注）
- 抖音分享链接解析

### v1.0.0-hotfix.1

- 修复插件加载失败：延迟数据目录初始化，避免模块导入时提前调用 `StarTools`
- 修复 AstrBot v4.26.5 兼容性：`@filter.on_message()` → `@filter.on_llm_request()`
- 修复文件路径变量导入绑定为 `None` 的问题，改为通过 `config` 模块动态访问
- 新增 Web 管理面板：状态监控、QR 扫码登录、Cookie 管理

### v1.0.0-hotfix.2

- 修复 QR 扫码 API：切换到 SSO 端点 (`sso.douyin.com`)，使用 `aid=2906`，支持 base64 图片显示
- 修复前端 bridge 未定义时的回退逻辑，自动使用原生 `fetch()`
- 修复 Web 面板 API 路径前缀问题
